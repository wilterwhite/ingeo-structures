# ACI 318-25 - 4.11-4.14 FUEGO, CONSTRUCCION Y EVALUACION

---

## 4.11 RESISTENCIA AL FUEGO

### 4.11.1
Los miembros de concreto estructural deben satisfacer los requisitos de proteccion contra fuego del codigo general de edificacion.

### 4.11.2
Cuando el codigo general de edificacion requiera un espesor de recubrimiento mayor al especificado en **20.5.1**, el mayor espesor gobierna.

> **Referencia**: ACI/TMS CODE-216.1 proporciona guia adicional sobre resistencia al fuego.

---

## 4.12 REQUISITOS PARA TIPOS ESPECIFICOS DE CONSTRUCCION

### 4.12.1 Sistemas de Concreto Precolado

| Seccion | Requisito |
|---------|-----------|
| 4.12.1.1 | Se permite disenar segun **ACI/PCI CODE-319** |
| 4.12.1.2 | Incluir condiciones de carga y restriccion desde fabricacion hasta uso final |
| 4.12.1.3 | Incluir efectos de tolerancias |
| 4.12.1.4 | Incluir fuerzas y deformaciones en y adyacentes a conexiones |
| 4.12.1.5 | Trayectorias de carga en plano continuas; trayectoria de acero donde haya tension |
| 4.12.1.6 | Distribucion de fuerzas perpendiculares establecida por analisis o ensayo |

### 4.12.2 Sistemas de Concreto Presforzado

| Seccion | Requisito |
|---------|-----------|
| 4.12.2.1 | Se permite disenar segun **ACI/PTI CODE-320** |
| 4.12.2.2 | Diseno basado en resistencia y comportamiento de servicio en todas las etapas criticas |
| 4.12.2.3 | Considerar efectos de deformaciones elasticas/plasticas, deflexiones, cambios de longitud, rotaciones |
| 4.12.2.4 | Considerar concentraciones de esfuerzos por presfuerzo |
| 4.12.2.5 | Considerar perdida de area por ductos abiertos antes de inyeccion |
| 4.12.2.6 | Se permiten tendones externos |

### 4.12.3 Miembros Flexurales de Concreto Compuesto

| Seccion | Requisito |
|---------|-----------|
| 4.12.3.1 | Aplica el Codigo segun definicion del Capitulo 2 |
| 4.12.3.2 | Disenar miembros individuales para todas las etapas criticas de carga |
| 4.12.3.3 | Disenar para soportar cargas antes del desarrollo completo de resistencia compuesta |
| 4.12.3.4 | Detallar refuerzo para minimizar agrietamiento y evitar separacion de componentes |

### 4.12.4 Sistemas de Concreto Simple Estructural

| Seccion | Requisito |
|---------|-----------|
| 4.12.4.1 | Diseno segun **Capitulo 14** |
