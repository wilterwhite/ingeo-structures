# 26.13 REGISTROS E INFORMES

## 26.13.1 Registros de Inspeccion

**Tabla 26.13.1 - Contenido de Registros**

| Categoria | Informacion |
|-----------|-------------|
| **General** | Fecha, hora, clima, ubicacion |
| **Materiales** | Proveedor, numero de lote, cantidad |
| **Concreto** | Resistencia, revenimiento, aire, temperatura |
| **Refuerzo** | Tipo, tamano, ubicacion, conformidad |
| **Deficiencias** | Descripcion, accion correctiva |

## 26.13.2 Informes de Ensayo

**26.13.2.1** Incluir:
- Identificacion del proyecto
- Fecha y ubicacion de muestreo
- Identificacion de mezcla
- Resultados de ensayos
- Conformidad con especificaciones

## 26.13.3 Retencion de Registros

**26.13.3.1** Periodo de retencion:
- Minimo 3 anos despues de terminacion del proyecto
- Segun requisitos del propietario o jurisdiccion
