# 26.14 CURADO DEL CONCRETO

## 26.14.1 Requisitos de Curado

**Tabla 26.14.1 - Metodos de Curado**

| Metodo | Descripcion |
|--------|-------------|
| Agua | Inmersion, rociado, telas humedas |
| Membranas | Compuestos curadores segun ASTM C309 |
| Vapor | Curado acelerado (prefabricados) |
| Encofrado | Mantener encofrados en su lugar |

## 26.14.2 Duracion del Curado

**Tabla 26.14.2 - Tiempo Minimo de Curado**

| Condicion | Tiempo Minimo |
|-----------|---------------|
| Concreto convencional | 7 dias |
| Concreto de alta resistencia temprana | 3 dias |
| Clima caliente (> 90F) | Segun ACI 305R |
| Clima frio (< 50F) | Segun ACI 306R |
| Clase de exposicion F o S | 7 dias minimo |

## 26.14.3 Temperatura del Concreto

**26.14.3.1** Durante curado:
- Mantener temperatura >= 50F (10C)
- Proteger de congelamiento hasta alcanzar 500 psi

**26.14.3.2** Clima caliente:
- Temperatura de colado <= 95F (35C)
- Considerar retardadores y control de temperatura
