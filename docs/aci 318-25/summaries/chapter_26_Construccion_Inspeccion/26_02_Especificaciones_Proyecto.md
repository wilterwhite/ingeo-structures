# 26.2 ESPECIFICACIONES DEL PROYECTO

## 26.2.1 Informacion Requerida en Especificaciones

**Tabla 26.2.1 - Contenido Minimo de Especificaciones**

| Categoria | Informacion Requerida |
|-----------|----------------------|
| **Concreto** | f'c, clase de exposicion, contenido de aire |
| **Refuerzo** | Grado, tipo, recubrimiento |
| **Anclajes** | Tipo, material, profundidad de empotramiento |
| **Postensado** | fpu, fpy, tipo de tendon, lechada |
| **Durabilidad** | Clases de exposicion (F, S, W, C) |

## 26.2.2 Concreto

**26.2.2.1** Resistencia a compresion especificada (f'c):
- Valor a 28 dias (o edad especificada)
- Condiciones de curado de especimenes

**26.2.2.2** Tamano maximo del agregado

**26.2.2.3** Clases de exposicion y requisitos de durabilidad:
- Clase F (congelamiento-deshielo)
- Clase S (sulfatos)
- Clase W (contacto con agua)
- Clase C (corrosion)

**26.2.2.4** Contenido de aire para exposicion F1 o F2

**26.2.2.5** Proporcionamiento de mezcla segun 26.4.3

## 26.2.3 Refuerzo de Acero

**Tabla 26.2.3 - Especificaciones de Refuerzo**

| Item | Especificacion |
|------|----------------|
| Grado | 40, 60, 80, o 100 |
| Tipo | Corrugado, liso, malla soldada |
| Recubrimiento | Epoxico, galvanizado, inoxidable, o ninguno |
| Soldabilidad | Si se permite soldadura |
| Requisitos especiales | Ductilidad sismica, bajo carbono |

## 26.2.4 Postensado

**26.2.4.1** Especificar:
- fpu (resistencia ultima a tension)
- fpy (resistencia a fluencia)
- Tipo de tendon (toron, barra, alambre)
- Tipo de sistema (adherido, no adherido)

**26.2.4.2** Para tendones adheridos:
- Requisitos de lechada
- Proporcionamiento de lechada

**26.2.4.3** Para tendones no adheridos:
- Tipo de proteccion contra corrosion
- Requisitos de encapsulado

## 26.2.5 Anclajes

**26.2.5.1** Para anclajes colados en sitio:
- Material y grado
- Tipo de cabeza (cuadrada, hexagonal, deformada)
- Dimensiones

**26.2.5.2** Para anclajes post-instalados:
- Fabricante y modelo
- Informe de calificacion (ACI 355.2 o 355.4)
- Procedimiento de instalacion
