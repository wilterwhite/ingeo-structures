# 26.8 INSPECCION DE REFUERZO

## 26.8.1 Antes del Colado

**Tabla 26.8.1 - Lista de Verificacion de Refuerzo**

| Item | Verificar |
|------|-----------|
| Material | Grado y tipo correcto |
| Tamano de barra | Segun planos |
| Espaciamiento | Tolerancia +/- 3 in |
| Recubrimiento | Tolerancia segun 26.6.2 |
| Empalmes | Longitud y ubicacion |
| Ganchos | Dimensiones correctas |
| Amarres/soportes | Barras aseguradas |
| Limpieza | Sin oxido excesivo, aceite, suciedad |

## 26.8.2 Empalmes Mecanicos y Soldados

**26.8.2.1** Verificar:
- Tipo de empalme aprobado
- Instalacion segun fabricante
- Ensayos de calificacion disponibles

**26.8.2.2** Ensayos de produccion:
- Empalmes mecanicos: segun ACI 439.3R
- Empalmes soldados: segun AWS D1.4

## 26.8.3 Malla Soldada de Alambre

Verificar:
- Especificaciones correctas
- Integridad de soldaduras
- Traslapes adecuados
