# 26.3 PLANOS DE CONSTRUCCION

## 26.3.1 Informacion Requerida en Planos

**Tabla 26.3.1 - Contenido Minimo de Planos Estructurales**

| Categoria | Informacion Requerida |
|-----------|----------------------|
| **General** | Nombre del proyecto, ubicacion, fecha |
| **Materiales** | f'c, fy, fpu |
| **Cargas** | Cargas vivas, cargas de diseno especiales |
| **Geometria** | Dimensiones, elevaciones, secciones |
| **Refuerzo** | Tamano, espaciamiento, recubrimiento, detalles |
| **Anclajes** | Ubicacion, tipo, profundidad |
| **Juntas** | Ubicacion de juntas de construccion |

## 26.3.2 Detalles Tipicos de Refuerzo

**26.3.2.1** Mostrar:
- Longitud de desarrollo
- Longitud de empalme
- Ganchos estandar
- Dobleces

**26.3.2.2** Para elementos sismicos:
- Detallado de zonas de confinamiento
- Ganchos sismicos
- Empalmes restringidos

## 26.3.3 Planos de Taller (Shop Drawings)

**26.3.3.1** Requeridos para:
- Refuerzo de acero
- Postensado
- Prefabricados
- Anclajes y embebidos

**26.3.3.2** Contenido:
- Configuracion de barras
- Dimensiones de dobleces
- Ubicacion de empalmes
- Longitudes de corte
