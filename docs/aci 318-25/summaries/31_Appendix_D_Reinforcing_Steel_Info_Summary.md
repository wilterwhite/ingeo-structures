## FORMULAS UTILES

### Peso de Acero
```
Peso (lb) = 0.2836 * db² * L
```
donde db en pulgadas, L en pies

### Area de Barras por Pie de Ancho
```
As (in²/ft) = 12 * Ab / s
```
donde s = espaciamiento en pulgadas

### Cuantia de Refuerzo
```
rho = As / (b * d)
```
