## 26.1 ALCANCE

### 26.1.1 Aplicabilidad
Este capitulo aplica a:
- (a) Informacion requerida en documentos de construccion
- (b) Evaluacion y aceptacion del concreto
- (c) Inspeccion del concreto reforzado

### 26.1.2 Profesional Responsable del Diseno
El profesional de diseno (LDP - Licensed Design Professional) es responsable de:
- Preparar documentos de construccion
- Especificar materiales y tolerancias
- Definir requisitos de inspeccion

**26.2.4.1** Especificar:
- fpu (resistencia ultima a tension)
- fpy (resistencia a fluencia)

- Requisitos de lechada
- Proporcionamiento de lechada

**26.2.4.3** Para tendones no adheridos:
- Tipo de proteccion contra corrosion

- Material y grado

**26.2.5.2** Para anclajes post-instalados:
- Fabricante y modelo
- Informe de calificacion (ACI 355.2 o 355.4)

### 26.4.3 Proporcionamiento de Mezclas de Concreto

**26.4.3.1** Proporcionar mezcla para:
- Alcanzar f'c especificado
- Cumplir requisitos de durabilidad
- Cumplir requisitos de trabajabilidad

**26.4.3.2** Resistencia promedio requerida (f'cr):

**Tabla 26.4.3.2.1 - Resistencia Promedio Requerida**

| f'c (psi) | f'cr (psi) |
|-----------|------------|
| < 3000 | f'c + 1000 |
| 3000 - 5000 | f'c + 1200 |
| > 5000 | 1.10*f'c + 700 |

*Valores cuando no hay datos historicos de desviacion estandar*

**26.4.3.3** Con datos historicos (30+ ensayos):
```
f'cr = f'c + 1.34*s     (probabilidad 1 en 100 por debajo)
f'cr = f'c + 2.33*s - 500     (promedio de 3 consecutivos)
```
Usar el **mayor** de ambos valores.

## 26.9 INSPECCION DE POSTENSADO

### 26.9.1 Antes del Tensado

**Tabla 26.9.1 - Verificacion Pre-Tensado**

| Item | Verificar |
|------|-----------|
| Ductos/vainas | Ubicacion, perfil, integridad |
| Tendones | Tipo, cantidad, area |
| Anclajes | Tipo correcto, instalacion |
| Equipo de tensado | Calibracion vigente |
| Resistencia del concreto | f'ci alcanzado |

### 26.9.2 Durante el Tensado

**26.9.2.1** Registrar:
- Fuerza de gato
- Elongacion medida
- Secuencia de tensado

**26.9.2.2** Verificar elongacion:
```
Elongacion medida = Elongacion calculada ± 7%
```

**26.9.2.3** Si elongacion fuera de tolerancia:
- Detener tensado
- Investigar causa
- Notificar al ingeniero

### 26.9.3 Inyeccion de Lechada

**Tabla 26.9.3 - Verificacion de Lechada**

| Item | Requisito |
|------|-----------|
| Resistencia de lechada | Segun especificaciones |

| Tiempo de inyeccion | Antes de 48 horas post-tensado |
| Presion de inyeccion | 75-100 psi tipico |
| Llenado completo | Verificar salida en respiraderos |

## 26.11 EVALUACION Y ACEPTACION DEL CONCRETO

### 26.11.1 Muestreo de Concreto Fresco

**26.11.1.1** Frecuencia de muestreo:
```
Minimo: 1 muestra por cada 150 yd³ o fraccion
Minimo: 1 muestra por dia de colado
Minimo: 5 muestras por clase de resistencia
```

**26.11.1.2** Muestreo segun ASTM C172

### 26.11.2 Ensayos de Concreto Fresco

**Tabla 26.11.2 - Ensayos de Concreto Fresco**

| Ensayo | Norma | Frecuencia |
|--------|-------|------------|
| Revenimiento | ASTM C143 | Cada muestra |
| Temperatura | ASTM C1064 | Cada muestra |
| Contenido de aire | ASTM C231 | Cada muestra (F1, F2) |
| Peso unitario | ASTM C138 | Cuando se requiera |

### 26.11.3 Especimenes de Ensayo

**26.11.3.1** Cilindros estandar:
- 6 x 12 in (150 x 300 mm)
- 4 x 8 in (100 x 200 mm) permitido

**26.11.3.2** Cantidad por muestra:
- Minimo 2 cilindros para resistencia a 28 dias
- Cilindros adicionales para edades tempranas si se requiere

**26.11.3.3** Curado de cilindros:
- Curado estandar en laboratorio (ASTM C192)
- Curado en campo (cuando se especifique)

### 26.11.4 Ensayo de Resistencia a Compresion

**26.11.4.1** Procedimiento segun ASTM C39

**26.11.4.2** Resultado de ensayo:
- Promedio de 2 o mas cilindros de la misma muestra
- Si un cilindro muestra defecto obvio, puede descartarse

## 26.12 CRITERIOS DE ACEPTACION

### 26.12.1 Aceptacion de Resistencia

**26.12.1.1** El concreto se considera aceptable si:

**(a)** Promedio de cualquier 3 ensayos consecutivos:
```
Promedio >= f'c
```

**(b)** Ningun resultado individual:
```
Resultado individual >= f'c - 500 psi     (f'c <= 5000 psi)
Resultado individual >= 0.90*f'c          (f'c > 5000 psi)
```

### 26.12.2 Concreto de Baja Resistencia

**26.12.2.1** Si criterios de 26.12.1 no se cumplen:
- (a) Investigar causas
- (b) Considerar ensayos adicionales
- (c) Evaluar adecuacion estructural

### 26.12.3 Investigacion de Concreto de Baja Resistencia

**Tabla 26.12.3 - Opciones de Investigacion**

| Opcion | Procedimiento | Norma |
|--------|---------------|-------|
| Nucleos | Extraer y ensayar nucleos | ASTM C42 |
| Ensayos no destructivos | Martillo de rebote, ultrasonido | ASTM C805, C597 |
| Prueba de carga | Cargar elemento en campo | 26.12.4 |

### 26.12.3.2 Criterio de Aceptacion para Nucleos

**Satisfactorio si:**
```
Promedio de 3 nucleos >= 0.85*f'c
Ningun nucleo individual < 0.75*f'c
```

### 26.12.4 Prueba de Carga

**26.12.4.1** Se permite cuando:
- Nucleos no satisfacen criterios
- Existe duda sobre capacidad estructural

**26.12.4.2** Procedimiento:
- Cargar incrementalmente hasta carga de prueba
- Medir deflexiones
- Evaluar recuperacion

**26.12.4.3** Carga de prueba:
```
W_test = 0.85*(1.4*D + 1.7*L)
```

**26.12.4.4** Criterio de aceptacion:
```
Deflexion maxima <= l²/(20,000*h)
Recuperacion >= 75% de deflexion maxima (24 hrs)
```

## 26.14 CURADO DEL CONCRETO

### 26.14.1 Requisitos de Curado

**Tabla 26.14.1 - Metodos de Curado**

| Metodo | Descripcion |
|--------|-------------|
| Agua | Inmersion, rociado, telas humedas |
| Membranas | Compuestos curadores segun ASTM C309 |
| Vapor | Curado acelerado (prefabricados) |
| Encofrado | Mantener encofrados en su lugar |

### 26.14.2 Duracion del Curado

**Tabla 26.14.2 - Tiempo Minimo de Curado**

| Condicion | Tiempo Minimo |
|-----------|---------------|
| Concreto convencional | 7 dias |
| Concreto de alta resistencia temprana | 3 dias |
| Clima caliente (> 90°F) | Segun ACI 305R |
| Clima frio (< 50°F) | Segun ACI 306R |
| Clase de exposicion F o S | 7 dias minimo |

### 26.14.3 Temperatura del Concreto

**26.14.3.1** Durante curado:
- Mantener temperatura >= 50°F (10°C)
- Proteger de congelamiento hasta alcanzar 500 psi

**26.14.3.2** Clima caliente:
- Temperatura de colado <= 95°F (35°C)
- Considerar retardadores y control de temperatura

## 26.15 REMOCION DE ENCOFRADO

### 26.15.1 General

**26.15.1.1** No remover encofrado hasta que el concreto tenga resistencia suficiente para:
- Soportar cargas aplicadas
- Resistir danos por remocion

### 26.15.2 Tiempos Minimos de Remocion

**Tabla 26.15.2 - Tiempos Minimos (si no se verifica resistencia)**

| Elemento | Tiempo Minimo |
|----------|---------------|
| Muros verticales | 12 horas |
| Columnas | 12 horas |
| Costados de vigas | 12 horas |
| Fondo de losas (apuntalado) | 3 dias |
| Fondo de vigas (apuntalado) | 7 dias |
| Remocion de puntales | Segun analisis |

### 26.15.3 Reapuntalamiento

**26.15.3.1** Si se requiere remocion temprana de encofrado:
- Proveer reapuntalamiento
- Disenar reapuntalamiento para cargas de construccion
