## C.1 ALCANCE

### C.1.1 Aplicabilidad
Este apendice provee guia para:
- (a) Reduccion de impacto ambiental
- (b) Uso eficiente de materiales
- (c) Diseno para resiliencia
- (d) Durabilidad extendida

### C.1.2 Objetivos
- Reducir huella de carbono
- Promover economia circular
- Mejorar resiliencia ante eventos extremos
- Extender vida util de estructuras

## C.2 MATERIALES SUSTENTABLES

### C.2.1 Cementos Alternativos

**Tabla C.2.1.1 - Cementos con Menor Huella de Carbono**

| Tipo | Reduccion CO2 | Consideraciones |
|------|---------------|-----------------|
| Portland-caliza (PLC) | 10% | Similar desempeno |
| Portland-puzolana | 15-30% | Desarrollo lento |
| Portland-escoria | 25-50% | Color mas claro |
| LC3 (caliza+arcilla) | 30-40% | Emergente |

### C.2.2 Materiales Cementicios Suplementarios (SCMs)

**Tabla C.2.2.1 - SCMs Permitidos**

| SCM | Norma | Contenido Tipico |
|-----|-------|------------------|
| Ceniza volante | ASTM C618 | 15-35% |
| Escoria | ASTM C989 | 25-70% |
| Humo de silice | ASTM C1240 | 5-10% |
| Puzolana natural | ASTM C618 | 15-35% |

### C.2.3 Agregados Reciclados

**C.2.3.1** Agregado de concreto reciclado (RCA):
- Permitido hasta 30% del agregado grueso
- Absorcion mayor (4-8% vs 1-2%)
- Ajustar mezcla por absorcion

**C.2.3.2** Limitaciones:
- No usar en exposicion F2 o S2/S3
- Verificar reactividad alcali-silice
- Ensayar propiedades mecanicas

## C.3 OPTIMIZACION DE DISENO

### C.3.1 Uso Eficiente de Materiales

**Tabla C.3.1.1 - Estrategias de Optimizacion**

| Estrategia | Ahorro Potencial |
|------------|------------------|
| Optimizacion topologica | 10-30% concreto |
| Concretos de alta resistencia | 15-25% concreto |
| Postensado | 20-40% refuerzo |
| Losas aligeradas | 20-35% concreto |
| Prefabricados | Reduccion desperdicio |

### C.3.2 Diseno para Desconstruccion

**C.3.2.1** Principios:
- Conexiones desmontables
- Elementos reutilizables
- Documentacion para futuro

**C.3.2.2** Consideraciones:
- Evitar soldaduras permanentes cuando sea posible
- Usar conexiones atornilladas
- Modularidad de elementos

## C.5 RESILIENCIA

### C.5.1 Definicion

**C.5.1.1** Capacidad de:
- Absorber perturbaciones
- Adaptarse a cambios
- Recuperarse rapidamente
- Mantener funcionalidad

### C.5.2 Amenazas Consideradas

**Tabla C.5.2.1 - Amenazas y Mitigacion**

| Amenaza | Estrategia |
|---------|------------|
| Sismos | Ductilidad, redundancia |
| Viento extremo | Robustez, envolvente |
| Inundacion | Elevacion, materiales resistentes |
| Incendio | Proteccion, compartimentacion |
| Cambio climatico | Adaptabilidad, durabilidad |

### C.5.3 Diseno para Multiples Amenazas

**C.5.3.1** Principios:
- Identificar amenazas criticas
- Evaluar vulnerabilidades
- Priorizar intervenciones
- Considerar interacciones

| Recubrimiento incrementado | Proteccion del refuerzo |

**C.6.3.1** Sistemas de monitoreo:
- Sensores de corrosion
- Medicion de deformaciones
- Deteccion de fisuras
