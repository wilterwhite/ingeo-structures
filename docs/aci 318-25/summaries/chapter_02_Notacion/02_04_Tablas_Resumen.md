# TABLAS RESUMEN

## Tipos de Anclas

| Tipo | Caracteristica Principal |
|------|-------------------------|
| Colada | Instalada antes de colar concreto |
| Adhesiva | Usa adhesivo para transferir cargas |
| De expansion | Transferencia por apoyo directo o friccion |
| De rosca | Enganche mecanico de roscas con concreto |
| De socavado | Interbloqueo mecanico por socavado del concreto |

---

## Tipos de Concreto

| Tipo | Densidad | Caracteristica |
|------|----------|----------------|
| Peso normal | > 135 lb/ft³ | Agregados ASTM C33 |
| Ligero | 90-135 lb/ft³ | Agregados ligeros ASTM C330 |
| Ligero con arena | - | Agregado fino normal, grueso ligero |
| Todo ligero | - | Agregados fino y grueso ligeros |

## Clasificacion de Secciones

| Tipo de Seccion | Condicion de Deformacion |
|-----------------|-------------------------|
| Controlada por compresion | epsilon_t <= epsilon_ty |
| Transicion | epsilon_ty < epsilon_t < epsilon_ty + 0.003 |
| Controlada por tension | epsilon_t >= epsilon_ty + 0.003 |

## Resumen de Resistencias

| Simbolo | Descripcion |
|---------|-------------|
| Rn | Resistencia nominal |
| phi*Rn | Resistencia de diseno |
| U | Resistencia requerida |
| Ru | Efecto de cargas factorizadas |

### Relacion Fundamental

```
Resistencia de Diseno >= Resistencia Requerida
phi * Rn >= U
```
