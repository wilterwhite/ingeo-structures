# 2.2 SIMBOLOS GRIEGOS

## Simbolos Alfa (alpha)

| Simbolo | Descripcion |
|---------|-------------|
| alpha | Angulo que define la orientacion del refuerzo |
| alpha_1 | Angulo minimo entre refuerzo distribuido unidireccional y un puntal |
| alpha_c | Coeficiente que define contribucion relativa del concreto a resistencia de corte de muro |
| alpha_f | Relacion de rigidez a flexion de seccion de viga a rigidez de losa |
| alpha_fm | Valor promedio de alpha_f para todas las vigas en bordes de un panel |
| alpha_s | Constante usada para calcular Vc en losas y zapatas |
| alpha_sh | Factor de forma para definir limite de esfuerzo cortante para muros (Tabla 18.10.4.5) |

---

## Simbolos Beta (beta)

| Simbolo | Descripcion |
|---------|-------------|
| beta | Relacion de dimensiones largas a cortas |
| beta_1 | Factor que relaciona profundidad del bloque de esfuerzos rectangular a profundidad del eje neutro |
| beta_b | Relacion de area de refuerzo cortado a area total de refuerzo en tension |
| beta_c | Factor de modificacion de confinamiento para puntales y nodos |
| beta_dns | Relacion para reduccion de rigidez de columnas por cargas axiales sostenidas |
| beta_n | Factor para efecto del anclaje de tensores en resistencia efectiva a compresion de zona nodal |
| beta_s | Factor para efecto de agrietamiento y refuerzo de confinamiento en puntal |

---

## Simbolos Gamma (gamma)

| Simbolo | Descripcion |
|---------|-------------|
| gamma_f | Factor para determinar fraccion de Msc transferida por flexion en conexiones losa-columna |
| gamma_p | Factor para tipo de refuerzo de presfuerzo |
| gamma_s | Factor para determinar porcion de refuerzo en banda central de zapata |
| gamma_v | Factor para determinar fraccion de Msc transferida por excentricidad de corte |

---

## Simbolos Delta (delta, Delta)

| Simbolo | Descripcion | Unidades |
|---------|-------------|----------|
| delta | Factor de magnificacion de momento para curvatura del miembro | - |

| delta_s | Factor de magnificacion de momento para marcos no arriostrados | - |
| delta_u | Desplazamiento de diseno | in. |
| Delta_1 | Maxima deflexion durante primera prueba de carga | in. |
| Delta_2 | Maxima deflexion durante segunda prueba de carga | in. |
| Delta_cr | Deflexion calculada fuera del plano correspondiente a Mcr | in. |
| Delta_n | Deflexion calculada fuera del plano correspondiente a Mn | in. |
| Delta_o | Deflexion lateral relativa entre parte superior e inferior de piso | in. |
| Delta_r | Deflexion residual 24 horas despues de remover carga de prueba | in. |
| Delta_s | Deflexion fuera del plano por cargas de servicio | in. |
| Delta_u | Deflexion calculada fuera del plano por cargas factorizadas | in. |
| Delta_x | Deriva de diseno del piso x | in. |
| Delta_fp | Incremento de esfuerzo en refuerzo presforzado por cargas factorizadas | psi |

## Simbolos Epsilon (epsilon)

| Simbolo | Descripcion |
|---------|-------------|

| epsilon_ty | Valor de deformacion neta de tension para definir seccion controlada por compresion |

## Simbolos Theta (theta)

| Simbolo | Descripcion |
|---------|-------------|
| theta | Angulo entre eje del puntal, diagonal de compresion, o campo de compresion y la cuerda de tension |
| theta_y | Rotacion de fluencia, radianes (Apendice A) |
| theta_ye | Rotacion esperada de fluencia del miembro (Apendice B) |
